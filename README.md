# @bedrockio/model

Bedrock utilities for model creation.

- [Install](#install)
- [Usage](#usage)
- Schema
- Search
- Validation
- Access Control

## Install

```bash
yarn install @bedrockio/model
```

## Usage

```js
import { loadModelDir } = from '@bedrockio/schema';
await loadModelDir('path/to/definitions/')
```
